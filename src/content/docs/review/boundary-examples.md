# 边界表达示例

## 通用模板

```text
这部分我实际做的是 [真实职责]。
它背后的核心原理是 [原理]。
工程上我处理了 [实现细节]。
遇到的主要问题是 [问题]。
这个结论的边界是 [边界]。
```

## MLIR 工具链

可以说：参与模型转换、模型编辑、图优化和量化分析工具建设。

谨慎说：主导整套 AI 编译器设计。除非能讲清完整架构、pass pipeline、后端代码生成和 ownership。

## 投机采样

可以说：完成工程化验证和平台适配分析。

谨慎说：成功上线并显著提升吞吐。项目最终受平台条件限制，没有完整落地。

## Agent Memory

可以说：参与用户偏好、历史摘要、关系状态等记忆能力的方案设计和模块建设。

谨慎说：负责完整实时语音系统或完整文档问答系统。语音链路和文档知识库不是主要职责。

## 端侧部署

可以说：负责模型转换、输入输出对齐、推理结果校验和性能分析。

谨慎说：熟悉所有 NPU 底层编译器实现。除非能讲到对应平台编译和 runtime 细节。
